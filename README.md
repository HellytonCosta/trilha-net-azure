# Employee Management System  

Este é um projeto MVC desenvolvido com ASP.NET, utilizando Tailwind CSS para estilização. O sistema inclui um CRUD para gerenciamento de funcionários e uma tabela de logs que registra ações realizadas no sistema.  

## Tecnologias Utilizadas  
- ASP.NET MVC  
- Entity Framework  
- MS SQL Server  
- Tailwind CSS  

## Funcionalidades  
- **Gerenciamento de Funcionários**: Criar, ler, atualizar e excluir registros de funcionários.  
- **Registro de Logs**: Captura e exibe ações realizadas no sistema para auditoria.  

## Como Executar  
1. Clone o repositório.  
2. Configure a string de conexão no `appsettings.json`.  
3. Execute as migrações do Entity Framework (`dotnet ef database update`).  
4. Inicie o projeto pelo Visual Studio ou com `dotnet run`.  

### Tailwind CSS 
- Sendo honesto não foi uma boa experiência usando o tailwind, talvez ainda falte alguns ajustes.
- Não esqueça de executar o npm install para o tailwind.

#### English - US
# Employee Management System  

This is an MVC project developed with ASP.NET, using Tailwind CSS for styling. The system includes a CRUD for employee management and a log table that records actions performed in the system.  

## Technologies Used  
- ASP.NET MVC  
- Entity Framework  
- MS SQL Server  
- Tailwind CSS  

## Features  
- **Employee Management**: Create, read, update, and delete employee records.  
- **Log Recording**: Captures and displays actions performed in the system for auditing.  

## How to Run  
1. Clone the repository.  
2. Configure the connection string in `appsettings.json`.  
3. Run the Entity Framework migrations (`dotnet ef database update`).  
4. Start the project using Visual Studio or `dotnet run`.  

### Tailwind CSS  
- Honestly, the experience using Tailwind wasn't great; maybe some adjustments are still needed.  
- Don't forget to run `npm install` for Tailwind.
